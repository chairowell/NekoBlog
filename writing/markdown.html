什么是 Markdown？

Markdown 是一种标记语言，使用简单的符号来标记文本，就像你拿不同类型的笔在纸上，给文字做不同形式的标注一样。

不仅能通过标记来改变文本的样式，还能改变文章的结构和排版方式，让文章更加美观、更方便阅读。

更多介绍

他的标记语法非常简洁，可以让人们更多地关注文档内容本身而非排版。

Markdown 使用易读写的纯文本格式来编写文档，可与部分 HTML 标签混合编写，而且可以导出成 HTML、PDF 、Word 以及其他常用格式的文件。

因为简洁、高效、易读、易写的特点，Markdown 被大量使用，如 Github、Wikipedia 等网站。当然他也不仅限在网站上使用，像 VSCode、Typora 等软件也同样支持使用 Markdown 来编写你的文档。

引言

可千万别被 「标记」或「语言」 吓到，其实 Markdown 的语法十分简单，他不像是专业语言有复杂的语法规则。你只需要记住一些常用的方法，把他当成一支功能很多的画笔来使用就行了。

如果这是你第一次接触 Markdown，别担心，这只会花费你大约30分钟，我会带着你，跟着我一步步的学习 Markdown 的基本语法。

在学习的过程中，我会补充一些比较高级的使用方法，这可能会多花费一些时间，但你完全不必担心，因为这并不影响你正常使用它。当然你也可以选择跳过，在你后续有空余时间的时候再回来看看。

基本语法

1. 标题

标题是每篇文章的核心，也是文章内容的开端。

如果要创建一个标题，那么我们需要在标题文字前面加上 #（井号）。

# 的数量代表着标题的级别，# 的数量越少，表示该标题的级别也越高，对应显示的字体也会越大。

注意

# 后面必须有一个空格，否则 Markdown 不会识别为标题。

为什么标题有等级之分？

  • 设置分级标题的主要目的是为了清晰地显示文章的层次，更好地理解信息的层次和逻辑关系。

  • 不同等级的标题通常通过字体大小、颜色、间距等样式区分，形成视觉对比。这种设计引导读者注意力，避免信息平铺直叙造成的疲劳。

  • 如果你的文章发布在互联网上，还有几种种额外的作用：

    1. 更好的被机器（爬虫）识别，可以帮助搜索引擎更好地收录和索引文章。
    2. 无障碍阅读，屏幕阅读器可以识别标题，帮助人们快速定位到各个章节。

假如说，我们要创建一个三级标题， 那么，我们的写法应该是：

### 这是一个三级标题

标签分为1~6级别，写法如下：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

2. 段落

根据内容划分成的部分，称为一个段落。段落划分是为了使文章更具可读性和逻辑性

如果我们要创建段落，那么，我们需要在段落文字前后分别加上一行或多行 空行 。

这是一个段落。

这是一个另一个段落。


这是第三个段落。

对应的效果如下：

这是一个段落。

这是一个另一个段落。

这是第三个段落。

对应的 HTML 如下：

<p>这是一个段落。</p>
<p>这是一个另一个段落。</p>
<p>这是第三个段落。</p>

3. 换行

在 Markdown 中换行有两种方式：

  • 在行末输入两个或两个以上的 空格 ，然后回车，Markdown 就会自动换行。
  • 在行末输入 <br> 标签，<br> 标签会结束当前行，并在下一行开始新的一行。
这是一个段落，  
这行会自动换行。

这是一个段落，<br>
这行会在下一行开始。

对应的效果如下：

这是一个段落，
这行会自动换行。

这是一个段落，
这行会在下一行开始。

对应的 HTML 如下：

<p>这是一个段落，<br>这行会自动换行。</p>
<p>这是一个段落，<br>这行会在下一行开始。</p>

提示

  • 因为多空格自动换行也是通过 <br> 标签实现的，所以对应的 HTML 非常相似。

4. 列表

列表是组织信息的有效方式：

  1. 将长段落拆分为条目式信息，避免因冗长文字产生疲劳。
  2. 适合对并列、递进或从属关系的内容进行分类。

Markdown 为此提供了有序列表和无序列表。

4.1 有序列表

有序列表是指有先后顺序的列表，数字或字母序号会显示在列表的左侧。

如果我们要创建有序列表，那么，我们需要在列表项前面加上 1.、2.、3. 等数字或字母序号。

1. 第一项
2. 第二项
3. 第三项

对应的效果如下：

  1. 第一项
  2. 第二项
  3. 第三项

对应的 HTML 如下：

<ol>
  <li>第一项</li>
  <li>第二项</li>
  <li>第三项</li>
</ol>

4.2 无序列表

无序列表是指没有先后顺序的列表，符号 ●、○、■、◻ 会显示在列表的左侧。

拓展

1. 标题

快捷键（部分编辑器支持，如：Typora）：

  • <Ctrl> + [1-6] ：按下 Ctrl 键和对应数字，光标所在那一行会变为对应级别的标题格式。
  • <Ctrl> + 0 ：按下该组合快捷键，光标所在那一行会清除标题格式。

标题的另一种写法是：

使用 ===（等于号） 和 ---（减号）来创建标题，效果与 # 和 ## 相同。

如果我们要创建一个二级标题，那么，我们的写法应该是：

二级标题
---

一级标题和二级标题的写法如下：

===
